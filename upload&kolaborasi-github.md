### Upload ke Repositori Github
```
git clone "paste link yang disalin disini tanpa tanda kutip"
cd nama-folder
git init 
git remote add origin "paste link yang disalin disini tanpa tanda kutip"
git add .  
touch initial
git commit -m "pesan-commit"
git push -u origin master
```
### Kolaorasi ke Repositori Github orang lain
